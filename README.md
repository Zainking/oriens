# oriens
Use pug and stylus create your static website.

## Installation
```shell
npm install oriens -g
```

## Basic Usage
Create a oriens project:
```shell
oriens create [projectname]
cd [projectname]
```

In your oriens project root directory run:
```shell
oriens build
```
